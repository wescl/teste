GGrid

APP MOBILE

GGrid é um aplicativo projetado para conectar jogadores que estão à procura de alguem para jogar juntos. Com apenas alguns cliques, ao selecionar a plataforma e o jogo desejado, você pode iniciar sua busca. Quando alguém também procura por interesses semelhantes, a conexão está pronta. Simplificando, o GGrid facilita a busca por parceiros de equipe, proporcionando uma experiência ágil e eficiente.

Motivo da criação

Minha experiência como jogador inclui participação ativa em diversos grupos de jogos, onde encontrei muitos jogos cooperativos que precisa de interação multiplayer. Olhando os comentários, percebi que algumas pessoas ficaram comentando sobre a falta de companheiros para jogar. Desses comentários nasceu a ideia do GGrid. Este projeto procura preencher esta lacuna, fornecendo uma maneira para os jogadores comunicarem e desfrutarem de experiências mais divertidas e colaborativas.

Pagina inicial

Contém 4 rádios nos quais você pode selecionar a plataforma desejada, um seletor que é um modal com caixas de seleção contendo todos os jogos do banco de dados. Caso o jogo desejado não esteja disponível, o usuário pode enviá-lo para análise. Quando for aceito por algum administrador, será incluído no banco de dados mencionado no seletor. Ao realizar uma busca, o sistema realiza todas as verificações necessárias, como a compatibilidade entre o jogo e a plataforma, se o usuário já possui um chat ativo com você, entre outros.

Pagina de conversas

Contém as conversas em que o usuário logado está participando. Caso não haja nenhuma conversa, uma mensagem é exibida. O sistema apresenta um contador de mensagens não lidas e exibe a última mensagem enviada. Ao clicar na conversa desejada, o usuário é direcionado para o bate-papo correspondente, que inclui a foto do usuário participante, a qual pode ser clicada para visualização ampliada, opções para denunciar e apagar a conversa. Adicionalmente, há um íon-fab que permite ao usuário enviar seu @ de alguma plataforma previamente adicionada ao perfil. A conversa inclui um mecanismo de visualização e uma api de envio de notificações para casos em que o usuário não esteja ativo no aplicativo.

pagina perfil

Contém um banner selecionado por mim, pois meu plano do Firebase é o gratuito e tem limitações de espaço para implementar todas as funcionalidades desejadas. Inclui a foto do usuário, permitindo que ele faça alterações nas informações visíveis apenas para si. Apresenta os @ das plataformas caso haja, com um botão para editar o perfil, trocar informações básicas, sair da conta e um botão para apagar a conta.

pagina de adm

Contém 4 botões que direcionam para modais específicos, cada um com uma função diferente: Jogos em Análise: onde o administrador pode visualizar os jogos enviados pelos usuários que não encontraram o jogo desejado. Jogo: permite ao administrador remover jogos, se necessário. Usuários: fornece ao administrador informações sobre os usuários, permitindo alterações ou exclusões. Denúncias: possibilita ao administrador verificar os motivos das denúncias e revisar as conversas entre os usuários para chegar à melhor solução.
